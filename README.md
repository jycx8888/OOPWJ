This is a Java-based Assessment Feedback System to support academic evaluation and feedback management for multiple user roles, including administrators, academic leaders, lecturers, and students. The system features user authentication, class and module management, grading functionality, assessment design, feedback submission, and report generation. 

Step by step guide:
1) Download and extract the zip file.
2) If use Apache NetBeans IDE, please open the file from NetBeansProjects if not the code will not run. The project file suggest name as oopwj. (correct file path: NetBeansProjects/oopwj/src/main/java/oopwj)
3) Please use Visual Studio Code or Apache NetBeans IDE to test the application

Thank you for using the application
